## Running the project

The page works with a server hosted on port 8080. Just use your terminal and type

```console
cd legoWorkStation
```

```console
npm start
```

Then on your search engine type:

```console
localhost:8080
```


## Testing the project

I have used testing to allow for better maintainability.

```console
npm run test.only
```
